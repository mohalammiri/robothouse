## 🎮 الأوامر الصوتية الكاملة

### التفعيل والإيقاف
- **"روبوت استيقظ"** / **"robot wake up"** / **"hey robot"** → تفعيل
- **"توقف"** / **"نم"** / **"stop"** / **"sleep"** → إيقاف

### الوسائط
- **"اعرض صورة رقم 15"** → فتح صورة (1-140)
- **"اعرض الفيديوهات"** → قائمة الفيديوهات
- **"فيديو الشاورما"** → تشغيل فيديو محدد
- **"اغلق الصورة"** / **"اغلق الفيديو"** → إغلاق

### البحث على الويب 🆕
- **"ابحث في يوتيوب عن سيارات ذكية"**
- **"ابحث في جوجل عن أخبار الإمارات"**
- **"ابحث في ويكيبيديا عن الذكاء الاصطناعي"**
- **"افتح أفلام IMDB"**
- **"أخبار عن الاستثمار"**
- **"خرائط رأس الخيمة"**

## 🔧 واجهات JavaScript العامة (للمطورين)

```javascript
// فتح وسائط خارجية
window.rhOpenExternalVideo('https://youtube.com/watch?v=...', 'عنوان');
window.rhOpenExternalImage('https://example.com/img.jpg', 'صورة');

// البحث على الويب
window.rhSearchWeb('ابحث في يوتيوب عن سيارات');

// إدارة النوافذ
window.rhOpenInNewWindow(url, title);
window.rhCloseLastWindow();
window.rhCloseAllWindows();

// إعدادات الصوت
window.rhAudioMode('far'); // 'far', 'low', 'near', 'auto'
window.rhSetVoiceSpeed(1.2); // 0.5 إلى 2.0

// WhatsApp صامت
window.rhSendWhatsApp('رسالة', '962785353408');

// المحادثة
window.rhSaveConv();       // حفظ
window.rhRestoreConv();    // استرجاع
window.rhExportConv();     // تصدير

// الإحصائيات
window.rhGetKeyStats();    // حالة المفاتيح
window.rhSuggestIdeas();   // اقتراحات ذكية
window.rhShowHelp();       // فتح لوحة المساعدة
```

## 📝 ملاحظات مهمة

1. **الأمان**: مفاتيح API موجودة في الكود الأمامي (client-side). للإنتاج، يُنصح بنقلها إلى Backend وسيط.
2. **الخصوصية**: المحادثة تُحفظ في `localStorage` على جهاز المستخدم فقط.
3. **التوافق**: يعمل على Chrome/Edge/Safari/Firefox الحديثة (يحتاج Web Speech API).
4. **الميكروفون**: يتطلب HTTPS (أو localhost).
5. **WhatsApp**: الإشعارات الصامتة قد تتأثر بـ CORS — آلية fallback متوفرة.
